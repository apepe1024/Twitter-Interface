This app requires the use of your own config.js file in the root folder.
Template as follows:

`module.exports = {
  consumer_key:         '...',
  consumer_secret:      '...',
  access_token:         '...',
  access_token_secret:  '...',
  timeout_ms:           60*1000  // optional HTTP request timeout to apply to all requests.
};`
